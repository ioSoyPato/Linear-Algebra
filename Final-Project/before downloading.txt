# before downloading

It is very important that before running this ipynb document you download the document called ioSoyPato.ipynb 
from the Pato library repository, this document will do the same effect as a library and will allow you to access
the functions created by Patricio Villanueva that are very important to run the notebook. 

- The ioSoyPato library requires numpy version 1.24.3 or grater.
